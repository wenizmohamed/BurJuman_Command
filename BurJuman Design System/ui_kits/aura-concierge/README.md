# AURA Concierge — UI Kit

BurJuman.com consumer shopping assistant. A chat-shaped concierge surface that
keeps the institutional palette but speaks **second-person** ("you", "your") —
a deliberate exception to the board-copy rule.

This UI inherits all dark-mode surfaces, the gradient ribbon, AIHUBZ pill, and
Lucide icons. It only diverges from Command Center on tone of voice.

## Files

| File                  | What it is                                             |
|-----------------------|--------------------------------------------------------|
| `index.html`          | Mount + chat session + B Hub member context            |
| `SiteHeader.jsx`      | Slim BurJuman.com header with B Hub Gold chip          |
| `AuraHero.jsx`        | Gradient hero with the "Meet AURA" intro               |
| `ChatThread.jsx`      | Messages, AURA bubbles, user bubbles                   |
| `TenantCard.jsx`      | Inline rich result card — tenant / offer / fitting     |
| `Composer.jsx`        | Input field + suggestion chips                         |
